## nodeCMF
> 基于Nodejs+express+mongodb编写的一套内容管理系统，个人学习使用~

### 项目安装方式
- npm install 安装所有依赖的npm包
- 启动mongodb
- 在项目目录下，运行 node ./bin/www 
- 启动项目，在浏览器输入localhost:3000 即可访问
